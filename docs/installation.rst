Installation
============

Stable release
--------------

To install TRITON-SWMM Benchmarking, run this command in your terminal:

.. code-block:: sh

   uv add TRITON-SWMM_benchmarking

Or if you prefer to use ``pip``:

.. code-block:: sh

   pip install TRITON-SWMM_benchmarking


From source
-----------

The source files for TRITON-SWMM Benchmarking can be downloaded from the
`GitHub repository <https://github.com/lassiterdc/TRITON_SWMM_benchmarking>`_.

You can either clone the public repository:

.. code-block:: sh

   git clone git://github.com/lassiterdc/TRITON_SWMM_benchmarking

Or download the
`tarball <https://github.com/lassiterdc/TRITON_SWMM_benchmarking/tarball/master>`_:

.. code-block:: sh

   curl -OJL https://github.com/lassiterdc/TRITON_SWMM_benchmarking/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: sh

   cd TRITON_SWMM_benchmarking
   uv pip install .
